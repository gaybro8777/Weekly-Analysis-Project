About Project

Spotify Weekly Analysis is a program which shows the top ranked albums, tracks, singers of different genres. It uses the database of more than 4 weeks. You just need to select your favourite genre and the charts will show top songs of the week. Different programming languages are used to create this project. Python to acquire data and store into a database. JavaScript to display a dashboard visualization of the data using charts and graph. Html and CSS to built the website.
